# 单词训练

## 简介：

*百词斩*是我用来背单词的好工具，奈何本身太懒，复习周才开始背单词，再加上我没找到*百词斩*自定义词书的办法，就仿照*百词斩*的方式，利用deepseek做了这么一个网页。该网页页面现代化，符合大众审美，简洁无广告，纯本地化运行，附带部分实用功能，是大学生在复习周临时抱佛脚的好方法。

---

## 功能：

- 根据英文选择对应中文；
- 点击英文读出对应文本；
- 错词统计与薄弱词汇导出；
- 积分系统（单纯成就感）。

  更多功能正在规划，有意见请提issue。

## 补充说明：

仓库内已上传词汇表转换工具.py。大致注意事项如下：
- 需要python；
- 额外的库：pandas；
- 下载后python程序应与目标xlsx文件处于同一文件夹下；
- xlsx的A列必须是单元（如：Unit 1 XXX），B列必须是英文（如：apple），C列必须是中文（如：n. 苹果）；
- 转化后的csv需要放在仓库内同一文件夹下（与index.html保持一致）；
- 如果克隆到自己的仓库使用，建议修改html文件。

  *如果有问题就去问AI，能解决很多问题*
